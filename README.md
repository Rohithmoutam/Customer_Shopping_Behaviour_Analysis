# Customer Shopping Behavior Analysis

## Project Overview
Customer Shopping Behavior Analysis is a data analytics project that examines customer purchasing patterns, spending behavior, product preferences, and subscription trends using transactional data. The project leverages Python for data cleaning and exploratory analysis, SQL for business-driven insights, and Power BI for interactive dashboard visualization.

## Objectives
- Analyze customer shopping behavior and spending patterns.
- Identify high-value customer segments.
- Evaluate the impact of subscriptions, discounts, and shipping preferences.
- Discover top-performing products and categories.
- Generate actionable business recommendations.

## Dataset Information
The dataset contains 3,900 customer purchase records with demographic, transactional, and behavioral attributes, including:

- Customer Age and Gender
- Product Category and Item Purchased
- Purchase Amount
- Subscription Status
- Review Ratings
- Shipping Type
- Discount Applied
- Purchase Frequency
- Previous Purchases

## Project Workflow

### 1. Data Cleaning & Preprocessing
- Loaded dataset using Pandas.
- Handled missing values in review ratings.
- Standardized column names.
- Performed feature engineering.
- Validated data consistency.

### 2. Exploratory Data Analysis (EDA)
- Customer demographic analysis.
- Purchase behavior analysis.
- Spending trend analysis.
- Product category performance analysis.
- Subscription and loyalty analysis.

### 3. SQL Analysis
Business questions answered through SQL:

- Revenue by Gender
- High-Spending Discount Users
- Top-Rated Products
- Shipping Type Comparison
- Subscribers vs Non-Subscribers
- Discount-Dependent Products
- Customer Segmentation
- Top Products by Category
- Repeat Buyers Analysis
- Revenue by Age Group

### 4. Power BI Dashboard
Developed an interactive dashboard featuring:

- Customer Overview
- Revenue Analysis
- Category Performance
- Subscription Insights
- Sales Trends
- Age Group Analysis

## Key Insights
- Loyal customers contribute significantly to overall revenue.
- Subscription status influences customer retention and purchasing behavior.
- Certain product categories consistently generate higher sales.
- Express shipping customers tend to spend more per transaction.
- Young adults contribute the highest revenue among age groups.

## Business Recommendations
- Increase subscription adoption through exclusive benefits.
- Implement customer loyalty programs.
- Optimize discount strategies for profitability.
- Focus marketing efforts on high-revenue customer segments.
- Promote top-rated products in campaigns.
